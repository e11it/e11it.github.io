Сначала выдали модем Wi-Fi Модем Yota, а потом lu150
Но он не захотел работать на Mac OS X(после того как драйвера поставил Wi-Fi Модем Yota).

Решение:

Отключить yota modem.

Удаляем драйвера

cd /System/Library/Extensions
sudo kextunload /System/Library/Extensions/Yota1KYDriver.kext
sudo rm -rf /System/Library/Extensions/Yota1KYDriver.kext

Ребилдим кеш

sudo kextcache -system-prelinked-kernel
sudo kextcache -system-caches

Удаляем yota access

run applescript:

/Library/Yota/Yota\ Access/Uninstall Yota.applescript

reboot…
